# Vim Configuration

## Install
* Clone the repo with  
`git clone git@github.com:jonsowman/dotvim.git ~/.vim`  
* Install Vundle  
`git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/Vundle.vim`  
* Install bundles  
`vim +PluginInstall +qall`  
* Symlink vimrc  
`ln -s ~/.vim/vimrc ~/.vimrc`
